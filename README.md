# Clear — Android приложение

Небольшой сервис, который преобразует сложные медицинские и официальные документы в **понятные и структурированные инструкции** через последовательный процесс когнитивной адаптации.

### Как это выглядит:
Пользователь загружает документ:
- фотографирует бумажный документ,

Сервис находит и обрабатывает текст и показывает:
- упрощённую версию с простыми словами,
- структурированную карточку — список шагов,

## Технологии
- Kotlin
- Jetpack Compose (UI)
- Gradle

## Структура проекта (важные файлы)
- `app/src/main/java/.../ui/instruction/InstructionScreen.kt` — основной экран с Compose для инструкции
- `app/src/main/java/.../data/model/` — модели данных (шаги инструкции и т.д.)
- `app/src/main/res/` — ресурсы (строки, макеты, drawable)

(Полная структура проекта в корне — см. папку `app/`)

## Требования
- Android Studio (рекомендуется) или командная сборка через Gradle
- Установленный Android SDK и платформы, соответствующие конфигурации в `build.gradle.kts`

## Запуск локально
Откройте проект в Android Studio и запустите на эмуляторе или устройстве.

Команды для сборки из терминала (в корне проекта):

```bash
./gradlew assembleDebug
./gradlew installDebug   # если подключено устройство
```
